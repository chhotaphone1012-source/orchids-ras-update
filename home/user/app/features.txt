## API ROUTES (14 Total)
- POST/DELETE /api/seed - Seed/Clear database
- GET /api/games - Get all active games
- GET /api/games/[id] - Get single game
- GET/POST /api/leaderboard - Get/Add scores
- GET /api/users - Get all users
- GET /api/user-stats - Get user statistics
- GET /api/achievements - Get achievements
- GET /api/challenges - Get daily challenges
- GET /api/notifications - Get notifications
- GET /api/analytics - Get analytics data
- POST /api/auth/otp - Send OTP email
- POST /api/auth/otp/verify - Verify OTP
- GET /api/admin/otps - List active OTPs
- POST /api/admin/send-email - Send email to user
